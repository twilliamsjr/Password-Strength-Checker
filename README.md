# Password Strength Checker

## Overview
A Python-based cybersecurity tool that evaluates password complexity against industry-standard criteria. This script utilizes Regular Expressions (RegEx) to analyze input strings for length, case sensitivity, numeric values, and special characters.

## Features
- **Complexity Scoring:** specific criteria tracking (Length, Upper/Lower, Special Chars).
- **Real-time Feedback:** Provides actionable advice on how to improve weak passwords.
- **RegEx Implementation:** Demonstrates pattern matching for security validation.
